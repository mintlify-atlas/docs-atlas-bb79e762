Rappi2 is a production-ready logistics API built with FastAPI that manages the full delivery lifecycle — from order creation and driver assignment through GPS tracking, route planning, and financial reporting. It uses a hybrid persistence model: PostgreSQL handles transactional data (orders, users, payments) while MongoDB handles geospatial data, audit logs, and multimedia evidence.

## What Rappi2 does

Rappi2 covers all the operations a logistics platform needs:

- **Order management** — Create orders, track status through `Pendiente → En Proceso → En Tránsito → Entregado`, manage payments and invoices
- **Driver & vehicle operations** — Maintain a fleet of drivers and vehicles, assign them to orders with availability checks
- **GPS tracking** — Record real-time GPS pings, query driver proximity, and check geofence containment
- **Route planning** — Integrate with OpenRouteService to calculate optimal routes and automatically generate geofences along corridors
- **Security** — JWT access tokens (15-min expiry) with rotating refresh tokens, RBAC with resource/action permissions
- **Analytics** — Dashboard KPIs, SLA reporting, driver efficiency, sales trends, and geographic distribution
